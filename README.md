# Simulador-portafolio-de-inversiones-
Este proyecto consiste en el desarrollo de un simulador financiero en Python que permite gestionar y analizar la rentabilidad de un portafolio de inversión diversificado. El sistema integra activos de renta variable (acciones) y renta fija (bonos o CDTs), permitiendo simular operaciones de compra y venta, así como calcular la evolución del portafolio a lo largo del tiempo.

El simulador utiliza datos reales del mercado para obtener precios de cierre diarios, e incorpora factores clave como dividendos, intereses y comisiones de transacción.
